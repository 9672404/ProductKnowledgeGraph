# GoodsKG
GoodsKG, a knowledge graph that contains the product-product hierarchy producer sales goods relationship, which sum up to 1300 products and more than 90000 brands, 基于京东网站的商品上下级概念，商品品牌之间关系，商品描述维度等知识库，基于该知识库可以支持商品属性库构建，商品销售问答，品牌物品生产等知识查询服务，也可用于情感分析等下游应用．
